### Prerequisites

* Node.js

* npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/mafqoodaty.git

# Install server dependencies
cd server

# Install client dependencies
cd ../client
npm install
```

### Running the App

